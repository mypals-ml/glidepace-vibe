# Use Case: Connecting to a GitHub Project

This use case describes the step-by-step process of a user connecting to a GitHub project and viewing it as a Gantt chart.

**Preconditions:**

*   The user has a GitHub account.
*   The user has a GitHub project with tasks.

---

### Step 1: Accessing the Home Page

*   **View:** Home Page
*   **UI Elements:**
    *   Application Title
    *   "Connect to GitHub" button
*   **User Actions:**
    *   The user clicks the "Connect to GitHub" button.

---

### Step 2: Connecting to a GitHub Project

*   **View:** GitHub Connection View
*   **UI Elements:**
    *   "Project Name" search/select
    *   "Project ID" input field
    *   "Connect" button
*   **User Actions:**
    *   The user selects a GitHub project from the discovery list.
    *   The user enters the ID of the GitHub project.
    *   The user clicks the "Connect" button.

---

### Step 3: Viewing the Gantt Chart

*   **View:** Gantt Chart View
*   **UI Elements:**
    *   Gantt chart displaying project tasks
    *   Taskbars with start and end dates
    *   Dependencies between tasks
    *   Task groups
*   **User Actions:**
    *   The user can view the Gantt chart.
