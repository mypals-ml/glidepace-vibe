## Plan Review

- Independent axis clamping was the main risk because it can create a visible coordinate that is not on the curve.
- Sampling the existing cubic curve keeps the rendered control tied to the same geometry as the link path.
- Hiding the button when no sampled curve point fits inside the viewport avoids showing an affordance detached from an offscreen line.
- The component tests are the best regression layer because the behavior is coordinate-driven and deterministic.
