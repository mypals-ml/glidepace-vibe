# Implementation Plan: Task #175 — Default UI language

## Requirements

- Persist UI language in localStorage
- On first visit, detect browser language, apply it, and save
- When user changes language, save the new choice

## Implementation

1. Add `src/lib/uiLocaleStorage.ts` with detect/save/resolve/change helpers
2. Initialize i18n with `resolveInitialUiLocale()`
3. Route all language selectors through `changeUiLanguage()`
4. Add unit tests and update FEATURES.md
