# Google Jules AI Agent Instructions

This file contains specific instructions and protocols for the Google Jules AI agent working on the Glidelines project.

## Core Protocols

1.  **Read and Follow RULES.md**: Always adhere to the general development rules defined in [RULES.md](RULES.md).
2.  **Jules Labels**: *DO NOT* add a `JULES` label to any issue or task without explicit user confirmation.
3.  **Branching Strategy**:
    - Always create branches under the `/jules/` namespace (e.g., `jules/feature-name`).
    - If a branch is connected to a specific task or issue, the branch name must start with `task#{number}` (e.g., `jules/task#123-ui-fix`).
